## Exam Vim Configuration 

Just enter the following commands one by one in Vim command line 

(While in Vim, press ESC, press ":", enter the command, hit Enter)

```conf
# Enable the mouse input
set mouse=a

# Show lines numbers
set number

# Apply colors syntax
syntax on
```
